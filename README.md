<h1> Hello-my-friend </h1>

<p>>Вебзастосування "Привіт, мій друже!"

Вебзастосування виводить привітальне повідомлення "Привіт, мій друже!" по центру екрана.
Проєкт створено за допомогою HTML і CSS.
<p>Переглянути вебзастосування можна за гіперпосиланням:
  <a href="https://popovaak24-alt.github.io/Hello-my-friend/">https://popovaak24-alt.github.io/Hello-my-friend/</a>
</p>
<p>Для перегляду вебзастосування локально на комп'ютері послідовно виконайте наступні команди:</p>
<ol>
  <li><code>git clone https://github.com/popovaak24-alt/Hello-my-friend.git</code>, щоб клонувати репозиторій на локальний комп'ютер;</li>
  <li><code>cd hello-my-friend</code>, щоб перейти в папку hello-my-friend;</li>
  <li><code>start index.html</code>, щоб переглянути вебзастосування у браузері.</li>
</ol>
